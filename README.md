## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install --shamefully-hoist
```

### Starting the development server

Start the development server on [http://localhost:3000](https://localhost:3000)

```bash
npm run dev
```
